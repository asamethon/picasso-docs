## Picasso Kurulum
Belirtilen link üzerinden projeyi bilgisayarınızda indirip kod editörünüzde açın. [Picasso Github linki](https://github.com/papyon-apps/picasso-figma-plugin)

Projeyi editörümüzde açtıktan sonra kod editörünün terminalini açıp  `npm install` komutunu çalıştırıp projedeki paketleri indirmemiz gerekiyor. Bu komut çalıştıktan sonra `npm run build` komutuyla projenin buildini alıyoruz ve kullanıma hazır hale getiriyoruz.
